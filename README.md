# Automated SOC Infrastructure Deployment
This project automates the deployment of a Wazuh SOC stack using Ansible and Docker Compose. 

## Features
- Automated environment provisioning via Ansible.
- Containerized Wazuh Manager deployment.
- Optimized resource configuration for 4GB RAM environments.

## How to run
1. Clone the repo.
2. Run `ansible-playbook deploy-soc.yml -K`.
